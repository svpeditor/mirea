ОТЧЕТ

1. Массивы:
В Python массивы чаще реализуются через списки list, которые являются динамическими и допускают элементы разных типов. 
В C++ массивы статические по размеру, элементы одного типа, индексируются с нуля, и менять длину после объявления нельзя. 
В Java массивы объявляются через new, имеют фиксированный размер и тип, а работа с ними дополняется утилитами из класса Arrays.

2. Стеки:
Стек реализует принцип LIFO — элемент, добавленный последним, первым извлекается. 
В Python стек удобно строится на основе списка (append/pop). 
В C++ есть адаптер std::stack с методами push, pop, top, а в Java используется класс Stack или Deque с аналогичными операциями push, pop, peek.
